## Image guidelines

- Professional head-and-shoulders portrait, natural smile, clean/white background
- Square-ish crop (e.g. 600×600 or 800×800), face centered
- Optimised JPG/WebP, ideally under ~150 KB each
- **Use real photographs only, with student + parent consent.**

If a file is missing, the page automatically falls back to an on-brand
gradient monogram avatar — nothing breaks.
